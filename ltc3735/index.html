    LTC3735 双相 CPU 供电控制器¶

    数据来源: Linear Technology (ADI) LTC3735 数据手册 3735fa 原始数据手册 (PDF)

    概述¶

    LTC3735 是 Linear Technology(现属 ADI)针对 Intel Mobile CPU 供电 设计的双相同步降压控制器,采用恒定频率、峰值电流模式(current-mode)架构,驱动 4 只 N 沟道 MOSFET 组成的两路输出级。

    它面向 Intel IMVP-IV 规范,输出电压由 6-bit VID 数字接口直接编程,范围 0.7V ~ 1.708V(16mV 步进),同时提供 Boot 与 Deeper Sleep 两种电阻编程的输出电压,完整覆盖笔记本 CPU 从上电、Active、Deep Sleep 到 Deeper Sleep 的全部工作模式。

    两相输出以 180° 反相(antiphase / PolyPhase®)工作,把等效纹波频率翻倍,大幅降低输入/输出电容的 RMS 电流和纹波,是老一代笔记本主板 VCore 供电的经典方案。

    核心特性¶

    项目 参数
    输入电压范围 (VIN) 5V ~ 24V(外部 MOSFET 可扩至 28V)
    控制芯片供电 (PVcc) 4.5V ~ 7V
    输出电压 0.7V ~ 1.708V(6-bit VID)
    输出精度 ±1%
    VID 步进 16mV
    开关频率 210kHz ~ 550kHz(FREQSET 电压编程)
    相数 双相反相(180°)
    架构 恒频、峰值电流模式、同步 Buck
    拓扑 全 N 沟道 MOSFET,自举驱动
    保护 UVLO、OV、Foldback、短路 latch-off、软启动
    封装 36-Lead SSOP (0.209 宽) / 38-Lead QFN (5×7mm)
    工作温度 -40°C ~ 85°C

    引脚功能详解¶

    反馈与基准¶

    引脚 名称 类型 功能说明
    1 VFB 输入 误差放大器反馈输入,内部基准 0.6V
    5 VOA+ 输入 内部运放同相输入(用于 AVP/损耗补偿)
    6 VOA- 输入 内部运放反相输入
    7 OAOUT 输出 内部运放输出,SR = 5V/µs,可驱动 40mA RMS
    17 ITH 输入/输出 误差放大器输出、电流环补偿点,0 ~ 2.4V

    VID 与工作模式¶

    引脚 名称 类型 功能说明
    19~24 VID0 ~ VID5 输入 6-bit VID 逻辑输入,VL<0.3V / VH>0.7V,高阻,必须主动驱动
    4 PSIB 输入 Power Saving Indicator。高电平: 全同步高频模式;低电平: 省电模式(禁止反向电流)
    2 DPRSLPVR 输入 Deeper Sleep 状态输入。高: 进入 Deeper Sleep,由 RDPRSLP‖RDPSLP 决定输出;低: 退出
    8 STP_CPUB 输入 Deep Sleep 状态输入。低: 进入 Deep Sleep,输出在 VID 基础上偏移一个百分比(由 RDPSLP 决定)

    电阻编程点¶

    引脚 名称 类型 功能说明
    3 FREQSET 输入 频率设置,DC 0 ~ 2.4V 对应 210kHz ~ 550kHz(典型 1.2V→355kHz)
    14 RDPRSLP 输入 电阻到 VOA+,与 RDPSLP 并联决定 Deeper Sleep 输出电压
    15 RDPSLP 输入 电阻到 VOA+,决定 Deep Sleep 相对 VID 的偏移百分比
    18 RBOOT 输入 电阻到 VOA+,决定上电初期输出电压 VBOOT

    电流检测¶

    引脚 名称 类型 功能说明
    10 SENSE1+ 输入 通道 1 差分电流比较器 (+) 输入,共模范围 SGND ~ PVCC
    11 SENSE1- 输入 通道 1 差分电流比较器 (-) 输入
    12 SENSE2+ 输入 通道 2 差分电流比较器 (+) 输入
    13 SENSE2- 输入 通道 2 差分电流比较器 (-) 输入

    峰值电流阈值 VSENSE(MAX) 典型 72mV,即 RSENSE = 2 × (40mV / IMAX)(留有余量)。

    驱动器与自举¶

    引脚 名称 类型 功能说明
    33/30 TG1 / TG2 输出 顶管高端驱动,浮动摆幅 = PVCC 叠加在 SW 节点上
    27/25 BG1 / BG2 输出 底管低端驱动,0 ~ PVCC
    32/29 SW1 / SW2 输入 开关节点,顶管驱动的浮动地
    34/31 BOOST1 / BOOST2 电源 自举电容上极板,通过肖特基由 PVCC 充电
    28 PVCC 电源 驱动器与内部控制电路电源,4.7µF X5R/X7R 陶瓷 就近去耦到 PGND,严禁 >7V
    26 PGND 地 驱动器功率地,与 SGND 一点相连
    9 SGND 地 信号地,双通道共用,单点接 PGND

    状态与时序¶

    引脚 名称 类型 功能说明
    16 RUN/SS 输入 软启动电容 + 运行控制 + 短路超时定时器,VRUN/SS < 1V 关断,>1.5V 使能
    35 PGOOD 开漏输出 电源良好,输出在 ±10% 窗内为高阻,超出拉低,有 110µs 屏蔽保护过渡态
    36 MCH_PG 输入 MCH Power Good 输入,拉高前输出保持在 VBOOT;拉高后 15 个开关周期后 VID 才正式生效

    功能框图与工作原理¶

    主控制环:恒频、峰值电流模式¶

    • 每个通道以恒定频率工作,时钟置位 RS 锁存器 → 顶管开启;
    • 电感电流经 RSENSE 差分采样,主电流比较器 I1 检测到峰值达到阈值时复位锁存器 → 顶管关断、底管开启;
    • 峰值阈值由 ITH 引脚(误差放大器 EA 输出)决定;
    • VFB 与 0.6V 基准比较,负载增加 → VFB 略降 → ITH 升 → 平均电感电流升,直至匹配负载。

    两个通道时钟相差 180°,输入/输出侧的等效纹波频率翻倍,RMS 电流大幅降低,散热与瞬态双向改善。

    VID 电压设定¶

    • 内部一个 5.33k 的固定分压电阻 RATTEN 与外部由 VID0~5 控制的 RVID 组成分压器;
    • 输出电压 = 0.6V × (RVID + 5.33k) / RVID;
    • 16mV 步进,0.7V ~ 1.708V 共 63 档;
    • VID 逻辑输入是高阻,系统必须主动上下拉。

    上电与 Boot 电压¶

    • RUN/SS 被内部 1.5µA 缓慢拉升,达到 1.5V 后主环路使能,ITH 被钳位到最大值约 30%,随 CSS 继续上升被逐步释放;
    • MCH_PG 拉高之前,输出电压由 RBOOT 编程:

    VBOOT = 0.6V × R2·(R3+R5) / (R5·(R1+R2))

    • MCH_PG 拉高 15 个开关周期后 VID 才接管,期间 PGOOD 由 VBOOT 判定;
    • 若 RUN/SS 被拉低到 0,延时启动:tDELAY ≈ (1s/µF)·CSS,爬升:tIRAMP ≈ (1s/µF)·CSS。

    Deep Sleep / Deeper Sleep¶

    PSIB DPRSLPVR STP_CPUB 状态 输出电压
    High Low High Active(全同步) 6-bit VID
    Low Low High PSI(省电,禁反流) 6-bit VID,BG 反流关断
    Low Low Low Deep Sleep VID 减去 RDPSLP 编程的百分比偏移
    Low High X Deeper Sleep(Channel 2 关断) 由 RDPRSLP‖RDPSLP 决定,单相运行

    Deeper Sleep 下 Channel 2 完全关断("stage shedding"),消除一相的驱动与开关损耗,轻载效率大幅提升。

    反相(Antiphase / PolyPhase®)¶

    两相相差 180°,输入电流波形叠加后波谷抬升、波峰压低,输入 RMS 电流降幅可达 75%(相比单相),同时输出等效频率翻倍,COUT 需求也减半。

    PSIB 引脚模式切换¶

    PSIB 电压 模式
    > 0.6V 全同步:BG 允许持续导通,允许电感反向电流,瞬态最快
    < 0.6V 省电:检测到电感反流立即关断 BG,轻载效率高,但瞬态稍慢

    ⚠ 全同步模式下电感反流可能把能量泵回输入端,若输入电容 + 电源阻抗太大,可能将 VIN 抬升到危险电压 —— 数据手册明确警告 "BEWARE"。

    保护功能¶

    保护 阈值/条件 响应
    UVLO(PVCC) PVCC < 3.2V(下降) RUN/SS 被拉低,全部关断
    输出过压 (OV) VFB > 0.66V(+10%) TG 关断,BG 常开,直到 OV 解除
    Foldback 折返限流 VFB < 70%(输出跌落) 最大电流阈值降到约 32mV(≈ 满值 45%)
    短路 latch-off RUN/SS 电容放电到复位阈值 主控环路关断,直到 RUN/SS 被重新拉高
    短路检测 discharge VFB < 0.375V, VRUN/SS = 4.5V RUN/SS 以 1.5µA 电流放电
    过流 latch-off arming VRUN/SS 上升过 3.9V 后武装 之后条件满足即锁存
    latch-off defeat 向 RUN/SS 灌 >5µA 禁用短路 latchoff(仍保留 foldback)
    PGOOD 屏蔽 VID / STP_CPUB / DPRSLPVR 跳变时 PGOOD 被屏蔽 110µs,避免虚警
    MCH_PG 延迟 MCH_PG 上升沿后 输出锁在 VBOOT 再等 15 个周期

    典型应用电路¶

    数据手册 Figure 1 的典型应用:VIN 5 ~ 24V,VOUT 0.7 ~ 1.708V @ 40A,双相 1µH 电感 + 2mΩ RSENSE + 4 只 N-MOSFET。

    VIN 5~24V ─┬── CIN 10µF × 4 (35V)
           │
           ├── M1 (TG1) ─ SW1 ── L1 1µH ─┬── VOUT
           │              │              │
           │        M2 (BG1)              │
           │              │              │
           │             GND              │
           │                             CO 330µF/2V × 5
           ├── M3 (TG2) ─ SW2 ── L2 1µH ─┤
           │              │              │
           │        M4 (BG2)              │
           │              │              │
           │             GND              │
           │                              │
           └─────── (PVCC 4.5~7V, 4.7µF 就近去耦) ─── LTC3735

  MCH_PG / DPRSLPVR / STP_CPUB / PSIB / FREQSET → SoC/EC
  VID0~VID5 → CPU VID 接口
  RBOOT / RDPRSLP / RDPSLP → 电阻编程各状态电压
  BOOST1/2 → 0.47µF + BAT54A 从 PVCC 自举

    关键设计要点¶

    RSENSE 选择

    RSENSE = 2 × (40mV / IMAX),以 72mV 最大阈值 + 峰值(平均+半纹波)为依据,通常留 30% 余量。

    电感

    • 电感值同时影响纹波和瞬态,一般取 ΔIL ≈ 0.4·(IOUT/2);
    • 必须用铁氧体或 Kool Mµ,不能用低成本铁粉芯,否则铁损吃掉效率;
    • 铁氧体饱和是硬饱和,峰值电流一定要预留裕量。

    MOSFET

    • 必须 N 沟道,VGS = PVCC(4.5 ~ 7V),必须用逻辑电平门限管;
    • BVDSS ≥ VIN(通常 30V 档);
    • VIN:VOUT > 2:1 的移动 CPU 应用中,底管长时间导通 → 底管选 低 RDS(ON);顶管开关损耗为主 → 低 QG / 低 CRSS。

    CIN

    双相 CIN RMS 电流比单相少约 75%,但仍需按 D=VOUT/VIN 落在 (2k−1)/4 的最坏点选电容。

    COUT

    • ESR 上限:ESR < 4·RSENSE;
    • 容量下限:COUT > 1/(16·f·RSENSE);
    • OPTI-LOOP® 允许较宽的 ESR 与容量选择。

    PVCC 去耦

    4.7µF X5R/X7R 紧贴 PVCC/PGND 引脚,栅极电流的高 di/dt 直接靠这颗电容供,PCB 寄生电感必须控到最小。

    FREQSET

    必须用电阻分压主动偏置,不能悬空;不用改频时接 1.2V 得典型 355kHz。

    RUN/SS 电容 CSS

    同时决定软启动时长与短路 latch-off 超时:tDELAY = tIRAMP = (1s/µF)·CSS。要保守配合大电流负载浪涌。

    常见故障与排查思路¶

    现象 可能原因
    无 VCore,PGOOD 长期低 RUN/SS 电容对地漏电,或 UVLO(PVCC < 3.7V);检查 PVCC 及去耦
    上电有 VBOOT 但不跟随 VID MCH_PG 未拉高,或 VID 引脚未被主动驱动(高阻悬空)
    VCore 波动大 / 只有单相工作 一路 SENSE 电阻虚焊、或一路 MOSFET 门驱动异常;示波器看 SW1、SW2 相位是否 180°
    轻载啸叫,重载正常 PSIB 强制在低电平(省电模式)时电感/电容匹配问题;或 PSIB 电平抖动
    VIN 无故被拉高、烧 CIN 全同步模式(PSIB 高)下轻载反流泵电压,检查 PSIB 逻辑,或改用 PSIB=Low
    输出周期性打嗝复位 短路 latch-off 触发,CSS 过小或负载浪涌;必要时 RUN/SS 灌 >5µA 关闭 latch-off
    效率异常低 电感磁芯选错(铁粉芯铁损大);或 MOSFET 门电压不足(PVCC 偏低)

    💡 LTC3735 在 2005~2010 年一批 Intel Merom/Penryn 平台的笔记本主板上非常常见,现代主板已被更高集成度的 IMVP-6/7/8 IMON 单芯片取代,但作为学习 双相电流模式 + VID 数字调压 的教科书级案例仍然极具价值。
