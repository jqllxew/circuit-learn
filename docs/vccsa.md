# VCCSA — 系统代理供电

> 笔记待补充：Intel System Agent (SA) 供电轨分析，包括电压标准、供电来源、与 VCCIO 的关系。

## 概述

VCCSA（System Agent Voltage）为 Intel CPU 内部的系统代理模块供电，负责内存控制器、PCIe 控制器、显示引擎等关键 I/O 子系统的驱动力。VCCSA 电压直接影响内存超频稳定性和 PCIe 信号完整性。

## 关键参数

| 项目 | 典型值 | 说明 |
|------|--------|------|
| 电压范围 | 0.8V ~ 1.35V | 随内存频率增高 |
| DDR4 典型 | 1.05V ~ 1.2V | 2133MHz ~ 3200MHz |
| DDR5 典型 | 1.1V ~ 1.25V | 4800MHz ~ 5600MHz |
| 电流需求 | 2A ~ 8A | 取决于内存通道数 |

## 供电来源

```
VCCSA 可由以下之一产生：
1. VCore 经独立 LDO 降压
2. 独立 PWM Buck 从 19V/5V 产生
3. 与 VCCIO 共享前级、独立后级
```

## 与 VCCIO 的区别

| | VCCSA | VCCIO |
|--|-------|-------|
| 供电对象 | 内存控制器、PCIe/显示引擎 | I/O 缓冲器（DDR PHY） |
| 电压关系 | 通常 ≥ VCCIO | 通常 ≤ VCCSA |
| 失效症状 | 不认内存、不显示 | 内存报错、蓝屏 |
