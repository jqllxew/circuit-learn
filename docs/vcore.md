# VCore — CPU 核心供电

> 笔记待补充：此处存放 VCore 供电的详细分析，包括 PWM 控制器型号、多相并联结构、SVID/VR12 协议交互、电压动态调整机制。

## 概述

VCore 是笔记本主板上电流最大、设计最复杂的供电轨，为 CPU 核心提供稳定、可动态调节的电压。通常由多相同步降压控制器 + DrMOS 功率级构成。

## 关键参数

| 项目 | 典型值 | 说明 |
|------|--------|------|
| 电压范围 | 0.6V ~ 1.8V | 由 CPU SVID 动态调整 |
| 最大电流 | 30A ~ 100A+ | 取决于 CPU PL1/PL2 功耗 |
| 相数 | 2 ~ 6 相 | 多相并联均流 |
| 控制协议 | SVID (Intel) / SVI2 (AMD) | 串行电压调节协议 |
| 开关频率 | 300kHz ~ 1MHz | 每相 |

## 供电链路

```
19V 公共轨 → 多相 Buck 上管 Drain → 电感 → VCore 输出
                                   ↓
                              CPU 插座 (多引脚并联)
```
