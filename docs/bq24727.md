# BQ24727 充电控制器

> 数据来源: TI bq24727 数据手册
> [原始数据手册 (PDF)](assets/bq24727.pdf)
> SMBus 地址: 0x09H (7-bit: 0b00001001)
> 封装: WQFN-20 (RUK), 3.50 × 3.50 mm²

---

## 概述

BQ24727 是一款高度集成的高效同步降压电池充电控制器，专为笔记本电脑和移动电源系统设计。它采用精简的 20 引脚封装，提供核心的充电控制功能。

### 核心特性

| 项目 | 参数 |
|------|------|
| **输入电压** | 4.5V ~ 24V (绝对最大值 28V) |
| **充电电压** | 1.024V ~ 19.2V (16mV 步长, ±0.5% 精度) |
| **充电电流** | 最大 8.128A (64mA 步长, ±3% 精度) |
| **输入电流** | 最大 8.064A (128mA 步长, ±3% 精度) |
| **支持电池** | 1~4 节 Li-Ion |
| **开关频率** | 800kHz (固定) |
| **控制接口** | SMBus 1.2 |
| **待机电流** | 0.7mA |
| **充电拓扑** | 同步降压 |
| **温度范围** | -40°C ~ +85°C |

| 特性 | BQ24727 | BQ24780S |

## 引脚说明 (WQFN-20)

| 引脚 | 名称 | 类型 | 说明 |
|------|------|------|------|
| 1 | ACN | 输入 | 输入电流检测电阻负极输入。建议接 0.1µF 陶瓷电容到 GND 共模滤波，接 0.1µF 到 ACP 差模滤波。 |
| 2 | ACP | 输入 | 输入电流检测电阻正极输入。建议接 0.1µF 陶瓷电容到 GND 共模滤波，接 0.1µF 到 ACN 差模滤波。 |
| 3 | CMPOUT | 输出 | 独立比较器开漏输出。需要外接 10kΩ 上拉电阻。内部参考电压 0.6V/2.4V 可通过 SMBus 选择。当 CMPIN 高于参考电压时输出高电平。可通过在 CMPIN 和 CMPOUT 间接电阻设置迟滞。 |
| 4 | CMPIN | 输入 | 独立比较器输入。内置 50kΩ 串联电阻和 2MΩ 下拉电阻。可通过电阻分压连接到 IOUT 检测电流，或连接到 SRN 检测电池电压。 |
| 5 | ACOK | 输出 | AC 适配器检测开漏输出。当 ACDET > 2.4V 且 VCC > UVLO 时拉低，否则需要外接 10kΩ 上拉电阻。 |
| 6 | ACDET | 输入 | 适配器检测输入。通过电阻分压从适配器输入电压获得。当 ACDET > 0.6V 且 VCC > UVLO 时，REGN LDO、ACOK 比较器和 IOUT 激活。 |
| 7 | IOUT | 输出 | 适配器或充电电流缓冲输出。输出电压为检测电阻差分电压的 20 倍或 40 倍（可通过 SMBus 选择）。建议接 100pF 或更小的去耦电容到 GND。 |
| 8 | SDA | I/O | SMBus 开漏数据输入输出。按 SMBus 规范接 10kΩ 上拉电阻。 |
| 9 | SCL | 输入 | SMBus 开漏时钟输入。按 SMBus 规范接 10kΩ 上拉电阻。 |
| 10 | ILIM | 输入 | 充电电流限制输入。通过 3.3V 电阻分压设置。ILIM 电压或 DAC 限制电压中的较低值决定充电电流上限。当 ILIM < 75mV 时充电禁用，> 105mV 时充电启用。 |
| 11 | NC | - | 无连接 |
| 12 | SRN | 输入 | 充电电流检测电阻负极输入，同时用于电池电压检测。建议先接 7.5Ω 电阻再接到检测电阻，接 0.1µF 电容到 GND 滤波。 |
| 13 | SRP | 输入 | 充电电流检测电阻正极输入。建议先接 10Ω 电阻再接到检测电阻。 |
| 14 | GND | 电源 | IC 地。PCB 布局时连接到模拟地平面，仅通过底部 PowerPad 连接到功率地平面。 |
| 15 | LODRV | 输出 | 低侧功率 MOSFET 驱动输出。连接到 N 沟道 MOSFET 栅极。 |
| 16 | REGN | 输出 | 线性稳压器输出。6V LDO 输出，由 VCC 供电。当 ACDET > 0.6V 且 VCC > UVLO 时激活。接 1µF 陶瓷电容到 GND。 |
| 17 | BTST | 电源 | 高侧 MOSFET 驱动器电源。接 0.047µF 自举电容到 PHASE，接肖特基二极管从 REGN 到 BTST。 |
| 18 | HIDRV | 输出 | 高侧功率 MOSFET 驱动输出。连接到 N 沟道 MOSFET 栅极。 |
| 19 | PHASE | 输入 | 高侧 MOSFET 源极。连接到高侧 N 沟道 MOSFET 源极。 |
| 20 | VCC | 电源 | 输入电源，二极管或来自适配器或电池电压。使用 10Ω 电阻和 1µF 电容滤波以限制浪涌电流。 |
| - | PowerPad | 电源 | IC 底部暴露焊盘。模拟地和功率地仅在此处星形连接。必须焊接到 PCB，并通过过孔连接到模拟地和功率地平面，同时作为散热垫。 |

## 适配器插拔时序

> ⚠️ BQ24727 同样**没有内置ACFET/BATFET驱动**，适配器和电池的切换时序由外部电路决定。芯片只提供ACOK信号作为适配器存在指示。

### 🔌 适配器插入时序

| 时间点 | 信号/事件 | 说明 | 时序参数 |
|--------|----------|------|----------|
| T0 | 适配器物理插入 | 适配器电压开始上升 | - |
| T1 | **ACDET > 0.6V** | 芯片唤醒，REGN LDO启动 | 无延迟 |
| T2 | **ACDET > 2.4V** | 满足ACOK条件 | - |
| T3 | **ACOK 去抖计时开始** | 首次插入固定去抖时间 | **2ms** |
| T4 | **ACOK 拉低** | 适配器存在指示有效 | T3 + 2ms |
| T5 | **SMBus 激活** | EC可以开始配置寄存器 | REGN稳定后 |
| T6 | **Buck 转换器启动** | 开始充电<br>条件：ILIM > 105mV + 寄存器已配置 | SMBus配置完成后 |
| T7 | **软启动完成** | 充电电流逐步上升到设定值 | 约10ms |

> ⚠️ 注意：如果不是首次插入（适配器曾经存在过），ACOK去抖时间会变成 **1.3s**（默认），可通过ChargeOption[15]位修改为2ms。

---

### 🔌 适配器拔出时序

| 时间点 | 信号/事件 | 说明 | 时序参数 |
|--------|----------|------|----------|
| T0 | 适配器物理拔出 | 适配器电压开始下降 | - |
| T1 | **ACDET < 2.4V** | 低于ACOK阈值 | - |
| T2 | **ACOK 立即拉高** | 无去抖延迟 | < **10µs** |
| T3 | **Buck 转换器停止** | 充电停止 | 立即 |
| T4 | **电源切换** | 外部电路切换到电池供电 | 取决于外部设计，通常数百µs |

> 💡 BQ24727的ACOK下降沿是**无去抖的**，这一点和其他芯片不同。拔出适配器时EC会立即得到通知。

---
