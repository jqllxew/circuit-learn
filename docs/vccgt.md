# VCCGT — 核显供电

> 笔记待补充：Intel CPU 集成显卡 (GT) 供电轨分析，包括电压标准、独立供电方案、与 VCore 的关系。

## 概述

VCCGT（Graphics Core Voltage）为 Intel CPU 内部集成显卡 (iGPU / GT) 提供独立供电。在带有独立显卡的笔记本中，此供电轨可能不启用；在仅使用核显的平台中，VCCGT 的功耗占比可达整颗 CPU 的 40% 以上。

## 关键参数

| 项目 | 典型值 | 说明 |
|------|--------|------|
| 电压范围 | 0.5V ~ 1.2V | 由 GPU 频率动态调整 |
| 轻载电压 | 0.5V ~ 0.7V | 桌面待机/2D 模式 |
| 满载电压 | 0.9V ~ 1.2V | 3D 游戏/渲染 |
| 电流需求 | 0A ~ 30A | 取决于 GPU 负载 |

## 供电架构

```
                     ┌→ VCore (CPU 核心)
主 PWM 控制器 ───────┤
                     └→ VCCGT (核显)
                         │
                    (VCCGT 可与 VCore 共享多相控制器，
                     单独分配 1-2 相给 GT 轨道)
```

## 与 VCore 的区别

| | VCore | VCCGT |
|--|-------|-------|
| 供电对象 | CPU IA 核心 | 集成显卡 |
| 电压调节 | 独立 SVID 地址 | 独立 SVID 地址 |
| 功耗特点 | 突发高负载 | 持续中高负载 |
| 掉电影响 | 直接死机 | 花屏、掉驱动 |
