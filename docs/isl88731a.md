# ISL88731A 充电控制器

> 数据来源: Renesas ISL88731A 数据手册 FN6738 Rev 3.00 (2011年6月8日)
> [原始数据手册 (PDF)](https://www.semiee.com/file2/4833e40a41ed95bf09dc9ea1010e60bd/Renesas/Renesas-ISL88731A.pdf)
> SMBus 地址: 0x12H (7-bit: 0b00010010)
> ManufacturerID: 0x0049H | DeviceID: 0x0001H
> 封装: 28 Ld TQFN, 5.00 × 5.00 mm²

---

## 概述

ISL88731A 是 Renesas（原 Intersil）推出的高度集成的锂离子电池充电控制器，通过 SMBus 系统管理总线编程。作为智能电池系统 (SBS) 中的智能电池充电器 (SBC)，能够自动限制 AC 适配器电流，实现充电功率调节。

采用同步整流 Buck 转换器拓扑，具备二极管仿真功能以提高轻载效率并防止系统总线升压。支持 1~4 节串联锂电池充电，最大充电电流可达 8A。集成 MOSFET 驱动器和自举二极管，减少外围元件数量并缩小布板面积。低失调电流检测放大器配合 10mΩ 检测电阻可实现高精度检测。充电终止电压精度可达 0.5%。

### 核心特性

| 项目 | 参数 |
|------|------|
| **适配器电压范围** | 8V ~ 28V |
| **充电电压** | 4.096V ~ 19.2V (16mV 步长, 0.5% 精度) |
| **充电电流** | 最大 8A (128mA 步长, 3% 精度) |
| **适配器电流** | 最大 11A (256mA 步长, 3% 精度) |
| **支持电池** | 1~4 节 Li-Ion |
| **开关频率** | 300kHz |
| **控制接口** | SMBus 2.0 |
| **充电拓扑** | 同步降压 |
| **工作温度** | -40°C ~ +85°C |

## 工作模式

### 1. 适配器检测
- DCIN > VDD 时，ACOK 变低，指示适配器存在
- 适配器移除后，ACOK 变高

### 2. 充电模式
- 恒流 (CC) 阶段: 充电电流保持设定值
- 恒压 (CV) 阶段: 充电电压保持设定值，电流逐渐减小
- 充电终止: 当充电电流降至设定值的 1/10 时终止

### 3. 适配器电流限制
- 当适配器电流超过 InputCurrent() 设定值时，自动降低充电功率以限制适配器电流
- 系统负载优先，充电电流根据剩余功率动态调节

## 保护功能

| 保护类型 | 阈值 | 响应 |
|----------|------|------|
| **输入过压** | DCIN > 30V | 关断所有驱动器 |
| **电池过压** | V(BAT) > 108% 设定值 | 停止充电，FAULT 输出低 |
| **电池欠压** | V(BAT) < 2.5V/节 | 预充电模式，充电电流限制为 1/8 设定值 |
| **充电过流** | CSSP-CSSN > 150mV | 逐周期限流 |
| **适配器过流** | ACSP-ACSN > 150mV | 逐周期限流 |
| **电池短路** | V(BAT) < 2.0V | 停止充电，FAULT 输出低 |
| **热关断** | Tj > 150°C | 关断所有功能，降至 130°C 恢复 |
| **看门狗** | 170s 未收到 SMBus 写操作 | 重置所有寄存器到默认值 |

## 适配器插拔时序

> ⚠️ ISL88731A **完全没有内置隔保FET驱动**，是三款芯片里硬件最简的设计。适配器和电池的切换时序100%由外部电路决定，芯片只输出ACOK状态信号。

### 🔌 适配器插入时序

| 时间点 | 信号/事件 | 说明 | 时序参数 |
|--------|----------|------|----------|
| T0 | 适配器物理插入 | 适配器电压开始上升 | - |
| T1 | **DCIN > UVLO** | 芯片内部偏置启动 | DCIN > 8V |
| T2 | **ACIN 比较器触发** | 内部适配器检测 | ACIN > 2.4V |
| T3 | **ACOK 拉低** | 适配器存在指示有效 | **无内置去抖**，取决于外部RC |
| T4 | **外部ACFET导通** | 由外部电路根据ACOK信号控制 | 典型 100µs ~ 1ms |
| T5 | **SMBus 激活** | EC可以开始配置寄存器 | DCIN稳定后 |
| T6 | **Buck 转换器启动** | 开始充电 | SMBus配置完成后 |
| T7 | **软启动完成** | 充电电流从128mA逐步上升 | 128mA/400µs 步进 |

> ⚠️ 重要：ISL88731A的ACOK引脚**完全没有去抖**，适配器插拔时会有毛刺。几乎所有参考设计都会在ACIN引脚外挂100nF电容实现约100ms的去抖延迟。

---

### 🔌 适配器拔出时序

| 时间点 | 信号/事件 | 说明 | 时序参数 |
|--------|----------|------|----------|
| T0 | 适配器物理拔出 | DCIN电压开始下降 | - |
| T1 | **ACIN < 2.4V** | 低于检测阈值 | - |
| T2 | **ACOK 立即拉高** | 无任何去抖延迟 | < **5µs** |
| T3 | **外部ACFET关断** | 由外部电路控制 | 典型 50µs ~ 500µs |
| T4 | **电源切换到电池** | 外部二极管或BATFET导通 | 取决于外部设计 |
| T5 | **Buck 转换器停止** | 充电停止 | 立即 |

> 💡 ISL88731A是三款芯片中**拔出响应最快**的，ACOK几乎零延迟，但代价是插入时必须依赖外部RC滤波，否则会误触发。

### 典型外部隔保电路设计

使用ISL88731A时，必须外接：
1. P沟道MOSFET作为ACFET
2. 100nF电容在ACIN引脚做去抖
3. 简单的比较器或三极管电路实现电源切换
4. 时序通常在100µs ~ 1ms级别，系统电压会有50~200mV的跌落
