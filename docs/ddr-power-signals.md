# 内存供电信号

DDR 内存供电系统的关键信号说明，以 DDR4 为例。

---

## 信号概览

| 信号 | 全称 | 电压（DDR4） | 作用 |
|:--|:--|:--|:--|
| **VDDQ** | 核心供电 | **1.2V** | 给 DRAM 颗粒的核心电路供电，负载最大 |
| **VTTREF** | 参考电压 | **VDDQ/2 = 0.6V** | 精确基准电压，给 VTT 稳压器提供目标参考 |
| **VTT** | 终端匹配电压 | **VDDQ/2 = 0.6V** | 地址/命令/控制线的终端电阻上拉，需要大电流驱动能力 |

---

## 供电链路

```
VDDQ (1.2V)
    │
    ├──→ DRAM 颗粒核心电路
    │
    └──→ 电阻分压 → VTTREF (0.6V，精确基准)
                        │
                        └──→ VTT 稳压器 → VTT (0.6V，大电流输出)
                                              │
                                              └──→ 地址/命令/控制线终端电阻
```

**关键点：** VTT 和 VTTREF 数值上相等（都是 VDDQ/2），但分工完全不同：

- **VTTREF** — 只管提供精确的 0.6V 基准，负载极小，精度要求高
- **VTT** — 要输出大电流驱动多条信号线的终端电阻，跟踪 VTTREF 的电压

---

## 各信号详解

### VDDQ（核心供电）

- 内存供电系统的源头
- DDR4 标准 1.2V，DDR5 降至 1.1V
- 由主板 VRM（电压调节模块）产生
- 如果 VDDQ 不正常，VTTREF 和 VTT 都没法工作

### VTTREF（参考电压）

- 通常由 VDDQ 经两个精密电阻分压得到
- 作用是给 VTT 稳压器提供一个精确的目标电压
- 一般会在 VDDQ 和 VTTREF 之间接电容滤波
- 测量时误差应该在 ±1% 以内

### VTT（终端匹配电压）

- 由 VTTREF 驱动的专用稳压器产生
- 连接地址、命令、控制信号的终端电阻上拉
- 电流需求大——每个信号线都有终端电阻
- 常见 VTT 稳压器：RT9045、UP0109P

---

## 测量与排查要点

1. **先查 VDDQ** — 如果 1.2V 不出，检查 VRM 输入（通常来自 5V 或 3.3V 待机）
2. **再查 VTTREF** — 正常情况下应该是精确的 VDDQ 的一半
3. **最后查 VTT** — 即使 VTTREF 正常，VTT 稳压器也可能损坏不输出

如果 VTT 缺失或不稳，会导致内存 training 失败，表现为：不跑码、卡内存灯、蓝屏等故障。
