## 概述

TPS51125 是 TI 推出的同步 Buck 控制器，专门设计用于笔记本电脑系统供电。它是笔记本主板上最常见的芯片之一，几乎所有 2010~2018 年的笔记本都能看到它的身影。

采用双路独立同步 Buck 架构，通常用于产生主板上的 5V 和 3.3V 系统主供电。支持 D-CAP™ 自适应导通时间控制，具有超快的负载瞬态响应，不需要外部补偿电路。

### 核心特性

| 项目 | 参数 |
|------|------|
| **输入电压范围** | 4.5V ~ 28V |
| **输出电压范围** | 0.75V ~ 5.5V |
| **输出精度** | ±0.8% |
| **开关频率** | 300kHz 固定 |
| **控制模式** | D-CAP 自适应导通时间 |
| **工作温度** | -40°C ~ +85°C |
| **封装** | 20 Pin TSSOP, 6.50 × 4.40 mm² |

## ⚠️ V5FILT 引脚深度说明

这是 TPS51125 最容易被忽略但最关键的一个引脚：

### V5FILT 的真实作用

V5FILT 不是普通的滤波引脚，它是 **内部 5V LDO 的反馈检测点**。

芯片内部的 VREG5 LDO 并不是直接从 VREG5 引脚取样，而是从 V5FILT 引脚取样。这个设计的目的是为了补偿 VREG5 输出到栅极驱动器之间的线损。

### 正确的连接方式

✅ **标准设计：**
- V5FILT 引脚通过一个 0Ω 电阻或者直接短接到 VREG5 输出端
- V5FILT 引脚必须接一个 1µF 的陶瓷电容到地，位置尽可能靠近引脚

❌ **绝对不能做的事：**
1.  不能悬空
2.  不能只接电容不接 VREG5
3.  不能接其他电压

### 故障现象

如果 V5FILT 引脚的连接出现问题：
- VREG5 输出电压会偏低，通常只有 3V ~ 4V
- 栅极驱动电压不足，MOSFET 导通电阻变大，严重发热
- 输出纹波巨大，甚至出现啸叫
- 芯片频繁进入过流保护

> 💡 这是维修 TPS51125 电路时最常见的隐性故障，90% 的"换了芯片还是不行"的问题，最后都是 V5FILT 引脚虚焊或者电容失效。

## 完整工作流程

![TPS51125 工作流程分析](https://bytep102.tos-ap-southeast-1.bytepluses.com/circuit-learn/tps51125-workflow.png)

> 完整启动时序共分为 5 个阶段，任何一个阶段卡住都会导致 3V/5V 输出异常。

### 阶段 1：电源就绪
1.  VIN 输入达到 5.5V 以上
2.  EN0 引脚被拉高到 0.5V 以上
3.  内部 LDO 开始工作，产生 VREG3、VREG5 和 REF 基准电压

### 阶段 2：配置读取
芯片读取所有配置引脚的状态：
- SKIPSEL：轻载工作模式选择
- TONSEL：开关频率设置
- VFB1/VFB2：输出电压分压值

### 阶段 3：PWM 启动
1.  ENTRIP1/ENTRIP2 引脚被拉高
2.  对应通道开始软启动
3.  上管导通给电感充电，下管导通给电感放电
4.  LL 引脚反馈开关节点状态

### 阶段 4：闭环调节
1.  输出电压通过 OUT 引脚检测
2.  输出电压通过 FB 引脚取样反馈
3.  D-CAP 控制器自动调节导通时间
4.  BST 自举电路通过 PWM 脉冲自动充电

### 阶段 5：电源就绪
1.  输出电压稳定 2ms 后，PGOOD 引脚释放
2.  两路都正常后，系统 PG 信号有效

## 工作原理

### D-CAP 控制模式

TPS51125 最核心的特点就是它的 D-CAP 控制模式：
- 不需要外部补偿网络
- 自适应导通时间，自动跟踪输入输出电压
- 负载瞬态响应极快，典型 <1µs
- 输出电容可以使用低 ESR 的陶瓷电容

这也是为什么它成为笔记本系统供电标准的原因：外围元件极少，调试简单，可靠性极高。

### 启动时序

1.  VIN 上升超过 4.5V，内部 LDO 开始工作
2.  VREG5 上升到 5V，VREG3 上升到 3.3V
3.  当 EN 引脚被拉高后，软启动开始
4.  输出电压以 1.2ms 斜率上升到设定值
5.  输出电压达到 92% 设定值后，PGOOD 被释放

## 保护功能

| 保护类型 | 阈值 | 响应 |
|----------|------|------|
| **输入欠压锁定** | VIN < 4.2V | 关断所有驱动器 |
| **输出过压** | VOUT > 115% 设定值 | 关断上管，强制打开下管 |
| **过流保护** | 可通过 ILIM 引脚设置 | 打嗝模式保护 |
| **热关断** | Tj > 145°C | 关断所有功能，120°C 恢复 |
| **欠压保护** | VOUT < 70% 设定值 | PGOOD 拉低 |

## 典型应用电路

### 标准 5V/3.3V 双路输出

```
公共点 ──┬──→ VIN
         ├──→ 自举电容 ──→ VBST1
         ├──→ 自举电容 ──→ VBST2
         │
         ├──→ 上管 Q1 ──→ 电感 L1 ──→ 5V 输出
         │            ↓
         └──→ 下管 Q2 ──→ GND

公共点 ──┬──→ 上管 Q3 ──→ 电感 L2 ──→ 3.3V 输出
         │            ↓
         └──→ 下管 Q4 ──→ GND
```

### 外围元件推荐

| 元件 | 数值 | 说明 |
|------|------|------|
| 电感 L1/L2 | 3.3µH | 饱和电流 ≥15A |
| 上管 Q1/Q3 | AO4407 | P 沟道 MOSFET |
| 下管 Q2/Q4 | AO4413 | N 沟道 MOSFET |
| 输出电容 | 4×22µF | X7R 陶瓷电容 |
| 输入电容 | 2×22µF | 陶瓷电容 |
| 自举电容 | 0.1µF | 陶瓷电容 |

## 常见故障与维修

| 故障现象 | 90% 概率的原因 |
|----------|--------------|
| 公共点短路，烧保险 | 下管 Q2/Q4 D-S 击穿 |
| 只有 3.3V 没有 5V | 上管 Q1 烧坏，或者 VREG5 LDO 失效 |
| 输出电压只有一半 | FB 分压电阻变质 |
| 输出纹波大，跳音 | 输出陶瓷电容失效 |
| 插上电源就掉电 | ILIM 电阻脱落，过流保护误触发 |

> 💡 TPS51125 是笔记本主板上故障率最高的芯片之一，绝大多数 3V/5V 异常的故障最终都会追到这个芯片或者它周围的 MOSFET。
